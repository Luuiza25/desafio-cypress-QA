# desafio-cypress-QA

Testes E2E com Cypress para o site lojaebac.ebaconline.art.br

## Instalação
npm install

## Rodar testes
npm run cy:open   # runner
npm run cy:run    # headless

## Cenários incluídos
- Login: sucesso e senha inválida
- Cadastro: válido e email duplicado
- Produtos: listagem, busca, filtro
- Carrinho: adicionar, remover, alterar quantidade
- Checkout: fluxo feliz e endereço incompleto
